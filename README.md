# Uniswap simple clone

### Preview

![](preview.gif)

### Initial setup

```sh
yarn
```

### Usage

```sh
yarn start
```

### Browser

Open [http://localhost:3000/pools](http://localhost:3000/pools) to view it in the browser.

## Tools used

- React
- React-DOM
- React-Router-DOM
- Context API
- Graphql
- Apollo
- Local Storage
- Styled Components
